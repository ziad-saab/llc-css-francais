##CSS3 Transform The `transform` property can be used to literally transform the element. You can skew it, scale it and few other things. Let's try adding just a slight `rotate` to it. The value for `rotate` is defined in degrees. Add this declaration to the Profile heading. Try using positive or negative numbers until you find one that you like! ``` .profile h2 { transform: rotate(-4deg); } ``` The heading should look similar to this: ![heading transform](framework/img/workshop/title-transform.png) More about `transform` [here](http://css-tricks.com/almanac/properties/t/transform/).
##:hover Pseudo Selector Let's try adding some changes on hover using the `:hover` [psuedo-class selector](http://www.w3.org/wiki/CSS/Selectors/pseudo-classes/:hover). Psuedo selectors need to be attached to an element with a colon (`:`) with *no* space. A common use for this selector is for links. Let's add a colour change on hover. ``` a:hover { color: #ffffff; } ``` The `:hover` pseudo selector can be added to any element. Make a change on hover for the "hello" heading, to update the `transform` property. The code will look like this: ``` .profile h2:hover { transform: rotate(4deg); } ```
##Class exercise: hover & transition Add the transition property to the hover effects. Your page should look similar to [example 9](project/examples/9-css3.html). ```css a { color: #2980b9; transition: all 0.5s ease; } a:hover { color: #ffffff; } .profile h2 { ... margin-bottom: 10px; transform: rotate(-4deg); transition: all 0.5s ease; } .profile h2:hover { transform: rotate(4deg); } ```
